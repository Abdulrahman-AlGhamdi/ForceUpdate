# ForceUpdate

Help the user to implement the force update of the application by just providing the APK link needed.

## Installation

### Repositories

```groovy
allprojects {
repositories {
maven { url 'https://jitpack.io' }
}
}
```
### Dependency
```groovy
implementation 'com.github.Abdulrahman-AlGhamdi:ForceUpdate:0.1.2'
```
