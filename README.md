# ForceUpdate

Help the user to implement the force update of the application by just providing the link needed

## Installation

### Repositories

```groovy
allprojects {
	repositories {
		maven { url 'https://jitpack.io' }
	}
}
```
### Dependency
```groovy
    implementation 'com.github.Abdulrahman-AlGhamdi:ForceUpdate:Tag'
```
